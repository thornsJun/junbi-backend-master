智能 BI 平台：

区别于传统的 BI，用户（数据分析者）只需要导入最最最原始的数据集，输入想要进行分析的目标（比如帮我分析一下网站的增长趋势），就能利用 AI 自动生成一个符合要求的图表以及结论。

后端技术选型：

1、Spring Boot

2、MySQL 数据库

3、MyBatis Plus 数据访问框架

4、消息队列（RabbitMQ）

5、AI 能力（Open AI 接口开发 / 星球提供现成的 AI 接口）

6、Excel 的上传和数据的解析（Easy Excel）

7、Swagger + Knife4j 项目接口文档

8、Hutool 工具库
